# Ocean Practice
A Minecraft Practice / Duels plugin

## Current Features
- 1v1 matches
- Unranked queues
- Kits
- Shared Arenas (for non-build kits)
- Stabdalone Arenas (for build kits)

## Kit Settings
- Boxing
- No hunger loss
- Description (in queue gui)
- Kit display name
- No health regen
